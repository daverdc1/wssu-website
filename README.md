# WSSU Website

Marketing site prototype for Winston-Salem State University, built with TanStack Start, React, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080).

## Build

```bash
npm run build
npm run preview
```

## Deploy

The project is configured for Vercel. Deploy with:

```bash
npx vercel deploy --yes
```

Live site: https://wssu-website.vercel.app
